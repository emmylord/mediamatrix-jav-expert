## ❓ What is mediamatrix-jav-expert?

mediamatrix-jav-expert is a tool designed to help you organize and enrich your adult video (AV) collection. It works as a plugin that automatically identifies AV codes in file names. Then, it collects information like titles, cover images, cast members, and ratings from JavDB, a respected online database. If you want, you can choose to integrate official metadata from DMM, a major Japanese content provider, to get more precise information.

This tool supports popular media management systems such as Jellyfin, Kodi, Plex, and other metadata managers. It creates metadata files in standard formats like NFO, which these programs use to display detailed information for your video files.

---

## 📋 Key Features

- Detects AV codes from your file names automatically.
- Retrieves detailed metadata, including titles, covers, cast, and ratings.
- Supports JavDB as the primary metadata source.
- Optional integration with DMM for official Japanese metadata.
- Compatible with media servers and managers like Jellyfin, Kodi, and Plex.
- Generates standard NFO and image files to enhance your media library.
